# 🧹 Data Cleaning

Before starting the analysis, several data quality checks were performed.

### ✔ Total Records

```sql
SELECT COUNT(*) AS total_rows
FROM retail_sales;
```

### ✔ Unique Customers

```sql
SELECT COUNT(DISTINCT customer_id) AS unique_customers
FROM retail_sales;
```

### ✔ Product Categories

```sql
SELECT DISTINCT category
FROM retail_sales;
```

### ✔ Check NULL Values

```sql
SELECT *
FROM retail_sales
WHERE
    transaction_id IS NULL
    OR sales_date IS NULL
    OR sales_time IS NULL
    OR customer_id IS NULL
    OR gender IS NULL
    OR age IS NULL
    OR category IS NULL
    OR quantity IS NULL
    OR price_per_unit IS NULL
    OR cogs IS NULL
    OR total_sales IS NULL;
```

### ✔ Remove Important NULL Records

```sql
DELETE FROM retail_sales
WHERE
    transaction_id IS NULL
    OR customer_id IS NULL
    OR quantity IS NULL
    OR price_per_unit IS NULL
    OR cogs IS NULL
    OR total_sales IS NULL;
```

# 📊 Business Questions & SQL Solutions

## Q1. Retrieve all sales made on **2022-11-05**

```sql
SELECT *
FROM retail_sales
WHERE sales_date = '2022-11-05';
```

---

## Q2. Retrieve Clothing transactions where quantity sold is greater than 3 during November 2022.

```sql
SELECT *
FROM retail_sales
WHERE category = 'Clothing'
    AND quantity > 3
    AND YEAR(sales_date) = 2022
    AND MONTH(sales_date) = 11;
```

---

## Q3. Calculate total sales for each product category.

```sql
SELECT
    category,
    SUM(total_sales) AS net_sale,
    COUNT(*) AS total_orders
FROM retail_sales
GROUP BY category
ORDER BY net_sale DESC;
```

---

## Q4. Calculate the net profit margin percentage for each category.

```sql
SELECT
    category,
    ROUND(
        SUM(total_sales - cogs) / SUM(total_sales) * 100,
        2
    ) AS profit_margin_percentage
FROM retail_sales
GROUP BY category
ORDER BY profit_margin_percentage DESC;
```

---

## Q5. Find the average age of customers who purchased Beauty products.

```sql
SELECT
    ROUND(AVG(age), 2) AS avg_age
FROM retail_sales
WHERE category = 'Beauty';
```

---

## Q6. Retrieve all transactions where total sales exceeded 1000.

```sql
SELECT *
FROM retail_sales
WHERE total_sales > 1000;
```

---

## Q7. Find the total number of transactions made by each gender within each category.

```sql
SELECT
    category,
    gender,
    COUNT(*) AS total_transactions
FROM retail_sales
GROUP BY category, gender
ORDER BY category;
```
---

## Q8. Find the total revenue generated by each gender within each category.

```sql
SELECT
    category,
    gender,
    SUM(total_sales) AS total_revenue
FROM retail_sales
GROUP BY category, gender
ORDER BY category, total_revenue DESC;
```

---

## Q9. Calculate the average monthly sales for each year.

```sql
SELECT
    YEAR(sales_date) AS year,
    MONTH(sales_date) AS month,
    ROUND(AVG(total_sales), 2) AS avg_sales
FROM retail_sales
GROUP BY YEAR(sales_date), MONTH(sales_date)
ORDER BY avg_sales DESC;
```

---

## Q10. Find the Top 5 Customers Based on Highest Total Sales.

```sql
WITH CustomerSales AS
(
    SELECT
        customer_id,
        SUM(total_sales) AS grand_total_sales,
        DENSE_RANK() OVER (
            ORDER BY SUM(total_sales) DESC
        ) AS rank_number
    FROM retail_sales
    GROUP BY customer_id
)

SELECT *
FROM CustomerSales
WHERE rank_number <= 5;
```

---

## Q11. Find the Number of Unique Customers Who Purchased Items from Each Category.

```sql
SELECT
    category,
    COUNT(DISTINCT customer_id) AS unique_customers_count
FROM retail_sales
GROUP BY category
ORDER BY unique_customers_count DESC;
```

---

## Q12. Find Customers Who Have Placed More Than 50 Orders.

```sql
SELECT
    customer_id,
    COUNT(transaction_id) AS total_orders
FROM retail_sales
GROUP BY customer_id
HAVING COUNT(transaction_id) > 50
ORDER BY total_orders DESC;
```

---

## Q13. Find the Top 3 Highest-Valued Orders Within Each Category.

```sql
WITH RankedProducts AS
(
    SELECT
        category,
        customer_id,
        total_sales,
        DENSE_RANK() OVER (
            PARTITION BY category
            ORDER BY total_sales DESC
        ) AS rank_num
    FROM retail_sales
)

SELECT *
FROM RankedProducts
WHERE rank_num <= 3;
```
